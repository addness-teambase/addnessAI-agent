## コードブロック

### インラインコード

`const greeting = "Hello, World!";`

### コードブロック（JavaScript）

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10));
```

### コードブロック（Python）

```python
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))
```

### コードブロック（TypeScript）

interface User {
